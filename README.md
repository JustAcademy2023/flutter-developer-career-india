## Skill Requirements

### Must-Have (For Fresher Roles)
- [ ] Dart — OOP, null safety, async/await, streams
- [ ] Flutter widgets — Stateless, Stateful, layout
- [ ] State management — Provider and/or Riverpod
- [ ] Firebase — Authentication, Firestore, Storage
- [ ] REST APIs — dio package, JSON serialization
- [ ] GoRouter — navigation and deep linking
- [ ] Git/GitHub — version control and portfolio

### Good-to-Have (For Mid-Level Roles)
- [ ] BLoC architecture
- [ ] Flutter animations — flutter_animate, AnimationController
- [ ] Local storage — Hive, Drift
- [ ] Testing — unit, widget, integration tests
- [ ] App Store + Play Store publishing
- [ ] Responsive layouts — LayoutBuilder, MediaQuery

---

## Portfolio Requirements

Minimum 3 deployable projects:
- [ ] E-commerce App — product listing, cart, Firebase auth
- [ ] Chat Application — Firebase Realtime, push notifications
- [ ] Fitness / Task Manager — Riverpod, local database (Hive)
- [ ] Weather App — REST API, JSON parsing, location services
- All code on GitHub with clean READMEs and live demo recordings
